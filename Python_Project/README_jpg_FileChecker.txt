################################################################
ReadMe File for jpg_FileChecker.py

################################################################

To run the code you must have files ready in a folder and have 
the path of the folder you are trying to access.

For example: I am trying to access my JPGs folder the path
I will use is 'C:\Users\aliss\OneDrive\Desktop\Mason\510\Hashing_Python\cfrs510hw3\File2Check\JPGs'

The program will ask you the following.

"Enter the directory of the folder:"

The User should enter: 
C:\Users\aliss\OneDrive\Desktop\Mason\510\Hashing_Python\cfrs510hw3\File2Check\JPGs

###################################################################

The program will scan the files within the folder and send the user
a text file labeled AlissaOutput.txt in the same folder you enter
the intial directory. 


Within the content of that text file will input all the files that
are JPG files and give the the following:
1. Name of the File
2. Directory of the File
3. Hash of the File
4. Mac Time

###################################################################

If a file that was in the folder does not show up in the output text
file then that file was not a JPG file.

###################################################################

At the end of the code the only thing that will print in the console
will be the time it took to execute the code and the directory of
where the text file will be located.
